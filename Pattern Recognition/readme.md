## 💡 使用建议

1. **课程**: 零基础有一定难度。两位老师讲课，向世明老师讲得不错，另一位一直在读PPT。。。
2. **作业**: 主要是理论和编程，理论部分课件上基本都能找到，编程部分善用GPT。
3. **考试**: 闭卷。本人零基础，感觉考试有一定难度，多刷往年题。

4. **tips**: 纯理论课程，有难度，慎选。
